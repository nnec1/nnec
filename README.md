# 🎓 سیستم مدیریت آموزشگاه (LMS)

یک سیستم کامل و حرفه‌ای برای مدیریت آموزشگاه‌ها، مدارس و مراکز آموزشی.

## ✨ امکانات

### 👑 پنل ریس سیستم

- مدیریت مدیران (ایجاد، ویرایش، حذف)
- گزارشات مالی جامع (روزانه/ماهانه/سالانه)
- تنظیمات پیشرفته سیستم
- پشتیبان گیری و بازنشانی سیستم

### 👔 پنل مدیر

- مدیریت شاگردان (ثبت، ویرایش، حذف، آیدی کارت، QR)
- مدیریت کارمندان (استاد، حسابدار)
- مدیریت صنف‌ها و دروس
- مدیریت فیس (وصول، بدهکاران، تاریخ انقضا، رنگ قرمز)
- گزارش حاضری استادان و شاگردان
- مدیریت نمرات (فردی و گروهی)
- اعلانات با آپلود فایل
- پاسخ به شکایات شاگردان

### 👨‍🏫 پنل استاد

- ثبت حاضری شاگردان (با هشدار فیس منقضی شده)
- ثبت نمرات (فردی و گروهی)
- مدیریت کارخانگی
- برنامه زمانی هفتگی
- غیرفعال‌سازی شاگرد
- ارسال امتیاز و شکایت به مدیر
- ویرایش پروفایل شخصی

### 👨‍🎓 پنل شاگرد

- مشاهده آیدی کارت و QR کد
- مشاهده حاضری (ماهیانه)
- مشاهده نمرات با نمودار
- مشاهده وضعیت فیس (کل، پرداخت شده، باقی مانده، تاریخ انقضا)
- ثبت شکایت به مدیر
- ویرایش پروفایل
